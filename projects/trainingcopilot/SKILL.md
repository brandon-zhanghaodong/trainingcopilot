# TrainingCopilot - AI培训助手

## 任务目标
- 本 Skill 用于：通过 AI 助手辅助完成培训相关的核心任务，包括课程设计、内容生成、资料分析、学习路径规划、效果评估和培训运营管理
- 能力包含：
  - 培训课程设计与内容生成
  - 培训资料分析与总结
  - 个性化学习路径规划
  - 培训效果评估与改进
  - 培训运营与后勤支持
  - 学员辅导与答疑
- 触发条件：
  - "帮我设计一个关于XXX的培训课程"
  - "分析这份培训资料，提取核心要点"
  - "为这个岗位规划学习路径"
  - "评估这次培训效果并提供建议"
  - "帮我写一封XX培训的通知邮件"
  - "为这次培训生成项目计划表"
  - "解答学员的学习疑问"

## 前置准备
- 依赖说明：无
- 文件准备：无

## 操作步骤

### 1. 培训课程设计与内容生成

当用户需要设计培训课程时：

1. **收集需求信息**
   - 培训主题和目标受众
   - 培训时长（总时长、单次时长）
   - 培训形式（线上/线下/混合）
   - 预期学习目标和关键知识点

2. **生成课程大纲**
   - 参考课程设计模板（见 references/course-design-templates.md）
   - 根据时长合理分配模块和章节
   - 设计学习目标和考核方式

3. **生成详细内容**
   - 讲义：结合理论知识和实践案例
   - 练习题：设计不同难度的题目（基础/进阶/挑战）
   - PPT大纲：为每个章节设计要点和视觉建议

4. **内容优化**
   - 确保逻辑清晰、层次分明
   - 添加案例和互动环节
   - 考虑不同学习风格的学员需求

### 2. 培训资料分析与总结

当用户提供培训资料需要分析时：

1. **资料读取与理解**
   - 识别资料类型（文档、视频字幕、PPT内容等）
   - 提取核心观点和关键信息

2. **结构化分析**
   - 提炼知识框架和逻辑结构
   - 识别重点、难点和易错点
   - 标注可实践的内容和方法

3. **生成总结文档**
   - 概览部分：核心主题和适用场景
   - 要点提取：分章节/主题整理关键信息
   - 行动建议：可落地的实践步骤

4. **质量控制**
   - 确保总结的准确性和完整性
   - 保持原始资料的逻辑结构
   - 突出实用性和可操作性

### 3. 个性化学习路径规划

当用户需要为特定学员或岗位规划学习路径时：

1. **需求评估**
   - 目标岗位或学习目标
   - 学员当前水平（初学者/进阶/专家）
   - 时间安排和学习偏好
   - 可用资源和支持系统

2. **路径设计**
   - 参考学习路径设计方法（见 references/learning-path-design.md）
   - 分阶段设计（入门/提升/精通）
   - 为每个阶段设定学习目标和评估标准

3. **资源推荐**
   - 推荐合适的学习内容（课程/书籍/实践项目）
   - 设计学习顺序和时间安排
   - 建议学习方法和实践方式

4. **跟踪与调整**
   - 设计阶段性检查点
   - 提供学习进度跟踪方法
   - 根据学习效果调整路径

### 4. 培训效果评估与改进

当用户需要评估培训效果时：

1. **数据收集**
   - 收集培训反馈（问卷调查、测试成绩、实践表现等）
   - 识别评估指标（知识掌握、技能提升、行为改变）

2. **效果分析**
   - 参考评估方法（见 references/assessment-methods.md）
   - 分析培训目标的达成情况
   - 识别成功经验和改进空间

3. **生成评估报告**
   - 整体评估：培训效果的总体评价
   - 细节分析：各维度表现（内容/讲师/方式/环境）
   - 对比分析：前后对比、与其他培训对比

4. **改进建议**
   - 针对性改进措施
   - 优化培训内容和方式
   - 建立持续改进机制

### 5. 培训运营与后勤支持

当用户需要培训运营和项目管理支持时：

1. **培训前准备**
   - 生成培训通知邮件（面向学员）
   - 生成讲师邀请函（面向讲师）
   - 创建培训项目计划表（甘特图形式）
   - 提供课前提醒邮件模板

2. **培训中支持**
   - 生成开场主持稿
   - 生成结束主持稿
   - 提供现场互动环节设计建议
   - 生成每日回顾邮件（多日培训）

3. **培训后跟进**
   - 生成感谢信与评估邀请邮件
   - 生成培训总结报告（向上级汇报）
   - 提供后续跟进计划

4. **运营清单与模板**
   - 参考运营清单（见 references/training-operations-checklist.md）
   - 使用标准化模板提高效率
   - 根据具体场景调整模板内容

### 6. 学员辅导与答疑

当学员提出学习疑问时：

1. **问题理解**
   - 明确问题的核心和背景
   - 识别学员的知识缺口

2. **提供解答**
   - 根据学员水平调整解释深度
   - 使用具体案例和类比说明
   - 提供可操作的解决方案

3. **延伸学习**
   - 推荐相关学习资源
   - 建议练习和实践方式
   - 鼓励深入探索相关主题

4. **效果确认**
   - 确认学员是否理解
   - 提供后续学习建议
   - 建立持续辅导机制

## 使用示例

### 示例 1：设计新员工入职培训课程
**功能说明：** 生成完整的新员工入职培训方案
**执行方式：** 智能体直接生成
**关键步骤：**
1. 收集公司信息、岗位要求、培训时长
2. 生成包含企业文化、岗位职责、技能培训的课程大纲
3. 设计各模块的详细内容和练习题
4. 提供培训实施建议

**输入示例：**
```
帮我设计一个为期3天的新员工入职培训课程，对象是软件工程师，需要包含：
- 公司文化介绍
- 技术栈和开发流程
- 团队协作工具使用
- 项目实践
```

### 示例 2：分析技术培训资料
**功能说明：** 深度分析技术文档，提炼核心知识点
**执行方式：** 智能体直接生成
**关键步骤：**
1. 用户提供培训资料
2. 提取知识框架和重点内容
3. 识别实践方法和案例
4. 生成结构化总结文档

**输入示例：**
```
分析这份关于微服务架构的培训资料，帮我：
- 提取核心概念和架构原则
- 整理关键技术栈
- 总结实施步骤和注意事项
- 提供学习建议
```

### 示例 3：规划产品经理学习路径
**功能说明：** 为特定岗位设计系统的学习路径
**执行方式：** 智能体直接生成
**关键步骤：**
1. 了解学员背景和目标
2. 设计分阶段学习计划
3. 推荐学习资源和实践项目
4. 提供评估方法和调整建议

**输入示例：**
```
我想转型做产品经理，有3年开发经验，帮我规划一个6个月的学习路径，
包括理论知识学习、技能实践和项目经验积累。
```

### 示例 4：生成培训通知邮件
**功能说明：** 快速生成专业的培训通知邮件
**执行方式：** 智能体直接生成
**关键步骤：**
1. 收集培训基本信息（主题、时间、地点、对象）
2. 生成结构化的通知邮件
3. 包含培训目标、安排、报名方式等关键信息

**输入示例：**
```
帮我写一封培训通知邮件，内容如下：
- 课程：《高效沟通技巧》
- 时间：2024年3月15日 14:00-17:00
- 地点：总部3楼培训室
- 对象：全体销售人员
- 讲师：李明老师（资深沟通专家）
```

### 示例 5：评估管理培训效果
**功能说明：** 分析培训数据，提供改进建议
**执行方式：** 智能体直接生成
**关键步骤：**
1. 收集培训反馈和测试数据
2. 分析培训目标达成情况
3. 识别优势和改进点
4. 生成评估报告和改进方案

**输入示例：**
```
我们刚完成了一期中层管理培训，收集了以下数据：
- 参训人数：30人
- 满意度调查：平均4.2分（5分制）
- 测试成绩：平均85分
- 请评估培训效果并提供改进建议
```

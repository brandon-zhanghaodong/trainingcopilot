## 概览

本文档为培训HR提供了一套完整的运营清单和实用模板，旨在简化培训项目管理中的事务性工作，提升效率和专业度。你可以直接使用或根据这些模板进行调整。

**核心价值：**
- **标准化**：确保培训运营各环节不遗漏。
- **效率化**：快速生成各类沟通文案和管理工具。
- **专业化**：提升培训项目的整体形象和体验。

## 培训运营全流程清单

### 阶段一：培训前 (T-30至T-1)

**1. 规划与启动 (T-30)**
- [ ] 明确培训目标与核心内容 (联动TNA)
- [ ] 确定培训对象与规模
- [ ] 初步确定培训时间与形式（线上/线下/混合）
- [ ] 组建项目团队（讲师、助教、后勤支持）
- [ ] 编制培训预算

**2. 讲师与内容准备 (T-25)**
- [ ] 确定并邀请内部/外部讲师 (使用**模板2**)
- [ ] 与讲师沟通课程大纲和内容
- [ ] 审核并最终确认培训教材/PPT
- [ ] 准备案例、练习题等辅助材料

**3. 宣传与报名 (T-20)**
- [ ] 发布正式培训通知 (使用**模板1**)
- [ ] 开放报名通道（问卷星、LMS系统等）
- [ ] 跟踪报名情况，及时沟通
- [ ] 截止报名，确认最终学员名单

**4. 后勤与物料准备 (T-10)**
- **线下:**
  - [ ] 预定并布置培训场地（桌椅、投影、白板、电源）
  - [ ] 准备签到表、名牌、文具
  - [ ] 印刷培训手册、讲义
  - [ ] 安排茶歇、午餐
- **线上:**
  - [ ] 确定并测试直播/会议平台（Zoom, Teams, 腾讯会议）
  - [ ] 准备在线互动工具（Miro, Mentimeter）
  - [ ] 创建在线签到链接

**5. 学员沟通 (T-3至T-1)**
- [ ] 发送课前提醒邮件 (使用**模板3**)
- [ ] 发放预习材料（如有）
- [ ] 建立学员沟通群（微信/钉钉）
- [ ] 解答学员疑问

### 阶段二：培训中 (T)

- [ ] 提前半小时检查场地/平台设备
- [ ] 引导学员签到
- [ ] 发放培训材料
- [ ] 讲师介绍与开场 (使用**模板6**)
- [ ] 监控培训节奏，协助讲师把控时间
- [ ] 组织互动环节，活跃气氛
- [ ] 收集学员的即时反馈和问题
- [ ] 引导茶歇与午餐
- [ ] 课程结束时进行总结，布置课后作业
- [ ] 组织满意度评估（发放问卷）

### 阶段三：培训后 (T+1至T+30)

- [ ] 发送感谢与评估邀请邮件 (使用**模板4**)
- [ ] 整理并分析满意度问卷数据
- [ ] 收集并批改课后作业（如有）
- [ ] 整理培训资料（讲义、录屏）并分享给学员
- [ ] 支付讲师费用及其他款项
- [ ] 撰写培训总结报告，向上级汇报
- [ ] 进行学习效果追踪（T+30），评估行为改变（联动评估模块）
